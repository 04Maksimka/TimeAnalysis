# Бары. Свечи

![[Pasted image 20260609211448.png]]

**OHLCV** — формат свечных данных: Open, High, Low, Close, Volume.
*Open* — точка входа, стоимость актива на начале сделки, при покупке
*Close* — точка выхода, стоимость актива по завершении сделки, при продаже
*High* — максимальная цена на интервале между покупкой и продажей
*Low* — соответственно, минимальная цена
*Volume* — объем актива

High и Low вместе отображают динамику актива в ходе сделки
